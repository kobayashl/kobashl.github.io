Hugo + GitHub Pagesでブログ作ってみた

※Window環境

   |  小林

技術系のアウトプットをしたくて作りました。
技術系以外にも調べたことをまとめたりいろいろ活用していこうと思います。


事前準備

  • Hugoのインストール
  • Gitのインストール
  • GitHubのアカウント作成

事前準備方法

  • Hugoのインストール

    まずは下記のサイトからHugoのzipファイルをダウンロードしてください

    https://github.com/gohugoio/hugo/releases
    例：hugo_0.xx.x_Windows-64bit.zip ※Windows64bitの場合
    ※Windowのbit数確認方法 WINDOWSキー + PAUSEキー システムの種類のxxビット


    ダウンロードしたHugoを解凍してPATHを通す

    hugo_0.xx.x_Windows-xxbit.zipを解凍してhugo.exeを任意のフォルダに置いて
    WINDOWSキー + PAUSEキー -> システムの詳細設定 -> 環境変数 に移動します
    システム環境変数のPathをダブルクリック -> 新規をクリックして下記の図のようにhugo.exeが格納されているファルダを指定します

    上記を行う(PATHを通す)ことでhugo.exeがどこからでも使えるようにしました


  • Gitのインストール

  • GitHubのアカウント作成

    以下のサイトで分かりやすく記載されているのでこちらの「公開鍵の登録」までをおこなってください
    https://qiita.com/manabu-watanabe/items/ecf1b434baf305adaa00


ブログを作成しよう


Git Hubにリポジトリを作成

GitHubで以下のリポジトリを作成してください

プロジェクトのソースコード管理用リポジトリ；username/blog.git
※blogでなくても大丈夫です
公開用リポジトリ：username/username.github.io
※usernameはアカウント名

hugoのプロジェクトを作成

まずは、ブログを作成するフォルダを作成してください。ここにプロジェクトが作られます
次に、コマンドプロンプト開き(WINDOWSキー + Rキー cmdと入力後にenterキー)
コマンドプロンプトで先ほど作成したプロジェクトに移動して、以下を実行 ※blogは任意の名前です

hugo new site blog

上記を実行することでhugoの新規プロジェクトを作成することができます

hugoのテーマ適用

以下から好きなテーマを選びsubmoduleとして登録する
https://themes.gohugo.io/
※以下のコマンドは、私が使用しているテーマの場合です

cd blog
git init #リポジトリ化
git submodule add https://github.com/halogenica/beautifulhugo.git themes/beautifulhugo #テーマを別レポジトリで管理する
cp -r themes/beautifulhugo/exampleSite/* . #テーマを適用

blog/config.toml変更する
blog/config.toml

baseurl = "https://username.github.io" #リポジトリの名前

ローカルでブログの確認

hugo server #ローカルでブログの確認ができる

http://localhost:1313/ にアクセスすることでブログを確認することができます

上記で問題ないことを確認してリポジトリに追加する

rm -rf public #ビルドファイルを削除
git submodule add -b master https://github.com/username/username.github.io.git public #公開ファルダを登録

git remote add origin https://github.com/username/blog.git #構築用フォルダを登録
git push origin master #リモートリポジトリ(GitHub)にコミット

デプロイとプッシュの自動化
deploy.sh

#!/bin/sh

# If a command fails then the deploy stops
set -e

# Commit changes.
msg="rebuilding site $(date)"
if [ -n "$*" ]; then
    msg="$*"
fi

# blogのコミット
printf "\033[0;32mblog updates to GitHub...\033[0m\n"

# Add changes to git.
git add .
git commit -m "$msg"

# Push source and build repos.
git push origin master

# publicのコミット
printf "\033[0;32mDeploying updates to GitHub...\033[0m\n"

# Build the project.
hugo # if using a theme, replace with `hugo -t <YOURTHEME>`

# Go To Public folder
cd public

# Add changes to git.
git add .
git commit -m "$msg"

# Push source and build repos.
git push origin master

上記スクリプトをblogフォルダに保存して実行

./deploy.sh "コミットメッセージ"

数分後https://username.github.io にブログが公開されます

以上、お疲れ様でした
