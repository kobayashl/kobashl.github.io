Raspberry Piを使用した無線LANルーターの作成方法

RaspBerry Piを無線LANルーターとして使用する方法を紹介します。

使用した物

  • RaspBerry Pi 4B 8G
  • Micro SDカード 32GB
  • USB Type-C ACアダプタ
  • RaspBerry Pi 4用アルミケース
  • LANケーブル 0.5m

Raspberry Piのセットアップ

1. OSのインストール

Raspberry Pi Imager
https://www.raspberrypi.org/software/

GUIは不要なので、RASPBERRY PI OS LITEを選択します。
書き込むSDカードを選択して、WRITEを押下し書き込み完了まで待ちます。


2. SSH 有効化 & Wi-Fi接続設定

書き込み終了後、SDカードのルートディレクトリに以下を行います。

ssh接続を有効にするため、sshフォルダを作成

cd /Volumes/boot ※bootは、ディスク名
touch ssh

Wi-Fiに接続させるため、wpa_supplicant.confファイルを作成

nano wpa_supplicant.conf

以下を記載

country=JP
ctrl_interface=DIR=/var/run/wpa_supplicant GROUP=netdev
update_config=1
network={
    ssid="wifi_name"
    psk="password"
}

3. パスワード変更、アップデート

SSH接続　※初期パスワードは「raspberry」です。

ssh pi@raspberrypi.local

パスワード変更

passwd

アップデート

sudo apt update && sudo apt full-upgrade -y && sudo apt autoremove -y && sudo apt clean

4. raspi-configで初期設定

sudo raspi-config

raspi-configのアップデート

8 Update  Update this tool to the latest version

ホストネーム設定　※必須ではない

1 System Options  Configure system settings
    S4 Hostname  Set name for this computer on a network

地域設定　※必須ではない

5 Localisation Options  Configure language and regional settings
    L2 Timezone     
        Configure time zone
            Asia > Tokyo
    L4 WLAN Country Set legal wireless channels for your country
        JP Japan

ファイルシステムの最適化

6 Advanced Options  Configure advanced settings
    A1 Expand Filesystem  Ensures that all of the SD card is available

ルーター設定

1. PPPoE設定

sudo apt-get install pppoe pppoeconf
sudo pppoeconf ※契約しているプロバイダーの情報を入力

2. プライベートIPアドレスの固定

sudo nano /etc/dhcpcd.conf

    最終行に以下を記載
    interface wlan0
    static ip_address=192.168.2.1/24

sudo nano /etc/network/interfaces

    以下に書き換え
    iface wlan0 inet static
    address 192.168.2.1
    netmask 255.255.255.0

3. アクセスポイント設定

wpa_supplicantとhostapdは共存出来ないため、wpa_supplicantを起動しないようにする
sudo systemctl disable wpa_supplicant

sudo apt install hostapd

sudo nano /etc/hostapd/hostapd.conf
    以下を記載
    interface=wlan0
    driver=nl80211
    hw_mode=g
    channel=2
    country_code=JP
    ieee80211d=1
    ieee80211h=1
    ieee80211n=1
    ht_capab=[SHORT-GI-20][DSSS_CCK-40]
    wmm_enabled=1
    macaddr_acl=0
    local_pwr_constraint=3
    spectrum_mgmt_required=1
    auth_algs=1
    wpa=2
    wpa_key_mgmt=WPA-PSK
    rsn_pairwise=CCMP
    ssid=raspi
    wpa_passphrase=**********

sudo nano /etc/default/hostapd
    DAEMON_CONF="/etc/hostapd/hostapd.conf"

マスク解除
sudo systemctl unmask hostapd

自動起動有効化
sudo systemctl enable hostapd

4. ポートフォワード設定

sudo nano /etc/sysctl.conf
    「＃」を削除
    net.ipv4.ip_forward=1

5. DHCP設定

sudo apt-get install isc-dhcp-server

sudo nano /etc/dhcp/dhcpd.conf
    以下を記載
    subnet 192.168.2.0 netmask 255.255.255.0 {
        range 192.168.2.10 192.168.2.20;
        option broadcast-address 192.168.2.255;
        option routers 192.168.2.1;
        default-lease-time 600;
        max-lease-time 7200;
        option domain-name "local";
        option domain-name-servers 8.8.8.8, 8.8.4.4;
        }

sudo nano /etc/default/isc-dhcp-server
    INTERFACESv4="wlan0"
    #INTERFACESv6=""

自動起動有効化
sudo systemctl enable isc-dhcp-server

6. ファイアウォール設定

基本的な設定
sudo iptables -P INPUT DROP
sudo iptables -P FORWARD DROP
sudo iptables -t nat -A POSTROUTING -o ppp0 -j MASQUERADE
sudo iptables -A INPUT -i lo -j ACCEPT
sudo iptables -A INPUT -m state --state ESTABLISHED,RELATED -j ACCEPT
sudo iptables -A FORWARD -i wlan0 -o ppp0 -j ACCEPT
sudo iptables -A FORWARD -m state --state ESTABLISHED,RELATED -j ACCEPT
sudo sh -c "iptables-save > /etc/iptables.ipv4.nat"

起動時適用
sudo nano /etc/rc.local
    exitの前に以下を記載
    iptables-restore < /etc/iptables.ipv4.nat

最後に再起動

sudo reboot

再起動後、他の端末から接続できるはず。。
