WindowsでBashを使う

WindowsでBashを使用する方法を紹介します。

1. Cygwinのダウンロード

以下のサイトからをダウンロードします。
https://cygwin.com/install.html

64bit用 - setup-x86_64.exe
32bit用 - setup-x86.exe


2. Cygwinのインストール

ダウンロードしたsetup-x86_64.exeを実行します。

次へ

インストール方法を聞かれますので、インターネットからインストールを選択

Root Directory：Cygwinのインストールディレクトリを選択します。
※C:/Cygwinなどドライブ直下にインストールする場合、注意のポップアップが出ます。

Install For：インストールユーザの選択

  • All Users：全てのユーザーにインストール
  • Just Me：現在のユーザーにのみインストール

Cygwinのパッケージのダウンロードフォルダの選択です。

インターネット接続方法の選択です
プロキシを通していなければ、Use System Proxy Settingsで大丈夫です。

パッケージのダウンロード先を選択します。

ダウンロードしたパッケージをインストールするかの選択です。
次へで大丈夫です。

Create icon on Desktop：デスクトップにショートカットを作成するか
Add icon to Start Menu：Windowのスタートメニューにショートカットを作成するか

3. Cygwin64 Terminalを実行

Cygwin/bin/mintty.exeを実行
初回起動時に.bash_historyなどの作成をしてくれるので起動だけさせます。


4. CygwinのPATHを通す

Windowキー + PAUSEキー -> システム詳細設定 -> 環境変数 -> システム環境変数のPathをダブルクリック -> 新規 -> Cygwin\binを追加

上記でCygwinのPATHを通すことでコマンドプロンプトからLinuxコマンドを実行することができます。

以上、お疲れ様でした。
